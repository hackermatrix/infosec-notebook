
## Instruction Set Arch

- Abstraction of the interface between software and hardware. i.e the instructions that a CPU understands.



- Reduced instruction set computers (RISC).
	- Keep instructions very very simple.( simple combined to create complex things.)
	- Seperate storage and read instructions.
	- Come with CPUs that have many registers available to them.
  VS
- Complex instruction set computers (CISC).
	- Adavance instructions.(do a lot of things at once ).
	- mostly have less no. of registers.
