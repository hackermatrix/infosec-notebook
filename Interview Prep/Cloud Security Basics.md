Cloud - Regions - Availability Zones

Does Cloudfront help in Ddos attack, yes it does. 


CASB is a security policy enforcement point between users and cloud services that provides visibility into **shadow IT, data loss prevention, threat protection, and compliance — across all cloud platforms,**  like email not just AWS.

![[Pasted image 20260312115808.png]]
![[Pasted image 20260312115846.png]]

Here's a practical example. Imagine an employee at your company opens Salesforce and tries to export 10,000 customer records to a CSV file and then upload it to their personal Google Drive. A firewall wouldn't catch this because both Salesforce and Google Drive are allowed websites. The admin laptop policies might not catch it either because the employee is using approved apps. But a CASB would catch it because it understands the _action_ — bulk export of customer PII followed by upload to an unsanctioned personal account — and it blocks it.
