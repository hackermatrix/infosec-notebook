# 🔑 In Assembly, Types = **Sizes**

### Common sizes on x86 / x86-64

| Name                 | Size              | Typical keyword |
| -------------------- | ----------------- | --------------- |
| **Byte**             | 8 bits (1 byte)   | `byte`, `db`    |
| **Word**             | 16 bits (2 bytes) | `word`, `dw`    |
| **Doubleword**       | 32 bits (4 bytes) | `dword`, `dd`   |
| **Quadword**         | 64 bits (8 bytes) | `qword`, `dq`   |
| **Tbyte** (ten-byte) | 80 bits           | for x87 floats  |
