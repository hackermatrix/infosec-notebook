# Class Notes 5130 (15-01-2026)

## Function Calling Convention (Stack Layout)

When a function like `add(a, b)` is called:

1. **`push b`**: The second argument is pushed first (stored at a **higher address**).
2. **`push a`**: The first argument is pushed next (stored at a **lower address**).

> [!NOTE]
> Arguments are pushed onto the stack in reverse order (Right-to-Left) in standard C calling conventions (cdecl).
